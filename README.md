Este projeto foi criado para praticar JavaScript, especialmente lógica de programação, manipulação do DOM, validações e eventos. </br>

O projeto inclui validações inteligentes de peso e altura, mensagens claras, cores dinâmicas baseadas no resultado e uma interface responsiva. </br>

Funcionalidades: </br>

Calcular automaticamente o IMC do usuário </br>
Exibir a classificação correspondente (abaixo do peso, normal, sobrepeso, obesidade…) </br>
Validação de peso mínimo/máximo </br>
Validação de altura mínima/máxima </br>
Exibição de mensagens de erro  </br>
Resultado com fundo colorido, variando conforme o nível do IMC </br>
Tabela explicativa das classificações no próprio HTML </br>
Layout responsivo (via media queries) </br> </br></br>

Tecnologias utilizadas: </br></br>

HTML5</br>

CSS3</br>

Media Queries para design responsivo </br>

JavaScript </br>
